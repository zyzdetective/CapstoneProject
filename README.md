# Quickstart

### Install requirements.
``` bash
pip install -r requirements.txt
```

### Start the server.

``` bash
python manage.py runserver
```

`For CSE machine:` 
  
``` bash
python3 manage.py runserver
```
### Open the Home Page.
Open this url in your browser: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
